# README

This is a simple product management tool. Built using React on Rails :muscle: :fire:

- Ruby version

  **ruby 2.5.1p57**

- Dependencies
 ```
   bootsnap (>= 1.1.0)
   byebug
   capybara (>= 2.15)
   chromedriver-helper
   coffee-rails (~> 4.2)
   jbuilder (~> 2.5)
   listen (>= 3.0.5, < 3.2)
   mini_racer
   puma (~> 3.11)
   rails (~> 5.2.1)
   react_on_rails (= 11.1.4)
   sass-rails (~> 5.0)
   selenium-webdriver
   spring
   spring-watcher-listen (~> 2.0.0)
   sqlite3
   turbolinks (~> 5)
   tzinfo-data
   uglifier (>= 1.3.0)
   web-console (>= 3.3.0)
   webpacker
 ```
- Install

  To install all gems run:
  ```
  bundle install
  ```

- Database creation

  This was built using Ruby on Rails built SQLite db.
  To create db simply run:
  ```
  bin/rails db:migrate
  ```
